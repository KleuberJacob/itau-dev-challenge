# itau-dev-challenge
Projeto feito para o desafio de programação do Itaú. 
A ideia é simples: criar uma API REST que recebe transações de usuários sem utilização de banco de dados, desenvolvendo endpoints pré especificados com possibilidade de visualizar estatísticas das transações realizadas nos últimos 60 segundos, podendo excluir todas transações existentes. Desenvolvido em Java com Spring Boot.
